# Calculator
The Calculator App is a user-friendly application designed to perform basic mathematical calculations quickly and accurately. It allows users to add, subtract, multiply, and divide numbers through a simple and intuitive interface.

## 1. Purpose
The purpose of this application is to help users perform everyday calculations without the need for manual computation. The app is suitable for students, teachers, office workers and anyone who needs a reliable calculator.

## 2. Features
* Addition of numbers
* Subtraction of numbers
* Multiplication of numbers
* Division of numbers
* Clear button to reset calculations
* User-friendly interface
* Instant calculation results

## 3. System Requirements
* A web browser such as Google Chrome, Microsoft Edge, Firefox or Safari
* Internet connection 

## 4. How the Application Works
1. The user enters numbers using the calculator buttons.
2. The user selects an arithmetic operation (+, -, ×, ÷).
3. The user enters the second number.
4. The user presses the equals (=) button.
5. The application displays the result on the screen.
6. The Clear (C) button can be used to reset the calculator.

## 5. Benefits
* Fast and accurate calculations
* Easy to use for beginners
* Reduces the risk of manual calculation errors
* Accessible from multiple devices

##6. Tools Used
* HTML (for structure)
* CSS (for styling)
* JavaScript (for functionality)
* Visual Studio Code (for development)
* Web Browser (for testing)

## 7. Conclusion
The Calculator App is a practical and efficient tool for performing basic mathematical operations. Its simple design and functionality make it useful for everyday calculations while providing accurate and immediate results.
